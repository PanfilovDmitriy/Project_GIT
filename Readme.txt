Проект - Обучение с учителем
Цель проекта:
- Разработать модель, которая предскажет вероятность снижения покупательской активности.
- Выделить сегмент покупателей, проанализировать его и предложить, как увеличить его покупательскую активность

В результате были подготовлены и изучены предоставленные данные. 
На них были обучены четыре модели: 
- KNeighborsClassifier, 
- DecisionTreeClassifier, 
- LogisticRegression,
- SVC
Для каждой из них подбирались гиперпараметры. Для оценки качества моделей использовалась метрика ROC-AUC.  Лучшей моделью оказалась - логистическая регрессия с С=2, которая показала метрику ROC-AUC 0.9.
